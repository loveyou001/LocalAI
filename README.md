# บอทสำหรับเกม Lust-Goddess Online (เวอร์ชัน 1.0)

บอทนี้จะช่วยค้นหาและเริ่มการต่อสู้กับมอนสเตอร์โดยอัตโนมัติ โดยใช้ระบบต่อสู้ Auto ของเกม

## สิ่งที่ต้องมีก่อนการติดตั้ง

- **Python 3:** ตรวจสอบให้แน่ใจว่าคุณได้ติดตั้ง Python 3 บนคอมพิวเตอร์ของคุณแล้ว

## วิธีการติดตั้ง

1.  **ดาวน์โหลดไฟล์ทั้งหมด:**
    - `bot.py`
    - โฟลเดอร์ `images` (ที่มีไฟล์ `attack_button.png` และ `end_battle.png` อยู่ข้างใน)

2.  **เปิด Command Prompt หรือ Terminal:**
    - ไปยังโฟลเดอร์ที่คุณได้ดาวน์โหลดไฟล์ทั้งหมดมา

3.  **ติดตั้ง Library ที่จำเป็น:**
    - พิมพ์คำสั่งด้านล่างนี้แล้วกด Enter เพื่อติดตั้งโปรแกรมที่จำเป็นทั้งหมด:
      ```
      pip install pyautogui opencv-python
      ```

## วิธีการใช้งาน

1.  **เปิดเกม:**
    - เปิดเกม Lust-Goddess Online ใน Web Browser และไปที่หน้าจอที่คุณต้องการให้บอททำงาน (หน้าที่มีมอนสเตอร์ให้โจมตี)

2.  **รันบอท:**
    - กลับไปที่หน้าต่าง Command Prompt หรือ Terminal
    - พิมพ์คำสั่งด้านล่างนี้แล้วกด Enter เพื่อเริ่มการทำงานของบอท:
      ```
      python bot.py
      ```

3.  **ปล่อยให้บอททำงาน:**
    - บอทจะเริ่มค้นหาปุ่ม "Fight" บนหน้าจอและคลิกเพื่อเริ่มการต่อสู้โดยอัตโนมัติ
    - **สำคัญ:** **ห้าม**ให้หน้าต่างอื่นมาบังหน้าต่างเกม และพยายามอย่าขยับเมาส์ในระหว่างที่บอทกำลังทำงาน

## วิธีการหยุดบอท

- กลับไปที่หน้าต่าง Command Prompt หรือ Terminal ที่บอทกำลังทำงานอยู่
- กดปุ่ม `Ctrl` + `C` พร้อมกันเพื่อหยุดการทำงานของบอท
